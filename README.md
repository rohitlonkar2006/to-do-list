# To-Do List (static)

This is a small static to-do list demo created for quick personal use.

Files
- `index.html` — the HTML page with the task list.
- `styles.css` — styling for the page.

Usage
- Open `index.html` in your browser (double-click or run `start ./index.html` in PowerShell).
- Toggle items by clicking the checkboxes. This demo uses only HTML+CSS; state is not persisted.

Next steps you might want:
- Add simple JavaScript to add/remove tasks and persist state to `localStorage`.
- Add a small build step or package if you want a richer app.
